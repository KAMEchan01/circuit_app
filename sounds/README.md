# CIRCUIT ワークアウトアプリ要件定義書 v2.0

## 1. プロジェクト概要

### 1.1 アプリケーション名
**CIRCUIT** - インターバルトレーニング支援Webアプリ

### 1.2 目的
- 効果的なサーキットトレーニングの提供
- ユーザーのフィットネス習慣化支援
- **Amazonアソシエイト審査対応および収益化**

### 1.3 ターゲットユーザー
- フィットネス初心者〜中級者
- 時短トレーニングを求める社会人
- 自宅トレーニング実践者

## 2. サイト構造（更新版）

### 2.1 現在の構造
```
index.html (ワークアウト実行画面)
├── /sounds/ (効果音ファイル)

```

## 3. 必須ページ仕様（Amazonアソシエイト審査対応）

### 3.1 プライバシーポリシー (/privacy/)
**優先度**: 最高
**内容**:
- 個人情報の収集・利用目的
- Cookie使用について
- Amazonアソシエイトプログラム参加について
- 第三者配信の広告について
- お問い合わせ先

### 3.2 利用規約 (/terms/)
**優先度**: 最高
**内容**:
- サービス利用条件
- 免責事項
- 著作権について
- 禁止事項
- 規約変更について

### 3.3 サイトについて (/about/)
**優先度**: 高
**内容**:
- サイト運営者情報
- サイトの目的・理念
- 運営者の経歴・専門性
- お問い合わせ先

## 4. 収益化ページ仕様

## 7. 既存機能仕様（維持）

### 7.1 効果音システム
**格納場所**: `/sounds/` ディレクトリ

**音源ファイル**:
- `start_exercise.mp3`: セット開始音
- `end_exercise.mp3`: セット終了音
- `countdown.mp3`: カウントダウン音（オプション）
- `workout_complete.mp3`: 完了音（オプション）

**仕様**:
- 形式: MP3推奨
- 長さ: 1-3秒程度
- 品質: 44.1kHz, 128kbps以上

### 7.2 ワークアウト機能
- 8セット構成のサーキットトレーニング
- 20秒運動 + 10秒休憩のインターバル
- 視覚的タイマー表示
- 音響フィードバック
