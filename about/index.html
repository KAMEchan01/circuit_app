サイトについて

🏎️ CIRCUITとは

CIRCUITは、F1レースをテーマにしたタバタ式サーキットトレーニングWebアプリです。たった4分間で効果的な全身運動ができ、毎日の「完走」を目指すことで楽しく運動習慣を身につけることができます。

名前に込められた3つの意味

  • 🏃‍♂️ サーキットトレーニング - タバタ式の効率的な運動
  • 🏎️ F1サーキット - 継続的な周回（習慣化）
  • ⚡ 電子回路 - 運動→食事→記録の循環する仕組み

💪 搭載されているエクササイズ

  • プッシュアップ - 上半身の筋力強化
  • シットアップ - 腹筋・体幹の強化
  • スクワット - 下半身全体の強化
  • ジャンピングジャック - 有酸素運動・全身運動
  • ロシアンツイスト - 腹斜筋・体幹ひねり
  • マウンテンクライマー - 有酸素・全身筋力

🎪 タバタ式トレーニングとは

タバタ式トレーニングは、田畑泉博士が開発した高強度インターバルトレーニング（HIIT）です：

  • 20秒間の高強度運動
  • 10秒間の休憩
  • これを8セット繰り返し
  • 総時間4分間で効率的な運動効果

✨ 期待できる効果

  • 有酸素能力の向上
  • 筋力・筋持久力の向上
  • 脂肪燃焼効果
  • 基礎代謝の向上
  • 時短での効率的運動

🏆 主要機能

⏱️ スマートタイマー機能

  • 準備時間5秒のカウントダウン
  • 運動20秒・休憩10秒の正確なタイミング
  • 視覚的で分かりやすい進捗表示
  • 次の運動の予告表示

📊 記録・管理機能

  • 連続完走日数の表示（ストリーク機能）
  • 摂取カロリー・消費カロリーの管理
  • 食事記録（朝食・昼食・夕食・間食）
  • 運動記録（タバタ・ランニング・ウォーキング・筋トレ）

🎮 ユーザビリティ

  • スマートフォン・タブレット・PC完全対応
  • インストール不要のWebアプリ
  • オフラインでも基本機能が利用可能
  • F1テーマの楽しいデザイン

📈 開発の経緯

🎯 コンセプト設計

「筋トレ習慣のない初心者が楽しく継続できるアプリ」を目標に、F1レースをテーマとした習慣化システムを設計

⚡ タバタ式採用

科学的根拠に基づく効率的なトレーニング方法として、4分間で効果が期待できるタバタ式を採用

🎨 UI/UX設計

初心者でも迷わず使えるシンプルなインターface設計と、モチベーション維持のためのゲーミフィケーション要素を実装

🚀 リリース・運用

GitHub Pagesでの無料公開を開始。ユーザーフィードバックを基にした継続的な改善を実施中

💝 私たちの想い

運動は「継続」が最も重要です。CIRCUITは、忙しい現代人でも「毎日4分だけなら続けられる」という体験を提供したいと考えています。

F1レースのように、毎日の「完走」を積み重ねることで、気がつけば運動が習慣になり、健康的な生活を送れるようになる。そんなサポートができるアプリを目指しています。
